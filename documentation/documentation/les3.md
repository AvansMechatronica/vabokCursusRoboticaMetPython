# Les 3: Programmeren van sensoren

In deze les leer je hoe je verschillende sensoren kunt gebruiken om gegevens te verzamelen uit de omgeving van je robot. We behandelen onder andere ultrasone sensoren, infraroodsensoren en kleurensensoren. Je zult leren hoe je deze sensoren kunt aansluiten op je robot en hoe je de gegevens die ze verzamelen kunt verwerken in je Python-programma's.
